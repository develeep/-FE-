### 과제
#### 미션1. switch case문 작성하기

`switch/case`문을 사용해서 조건문을 작성하세요.

생성한 `animal` 변수에 할당된 값이 `cat`일 경우에 '고양이'가 출력될 수 있도록 조건문을 알맞게 작성해주세요.

`case` 문은 dog, tiger, cat, lion으로 나눠서 작성하고,
`case`문의 내부에는 각각의 동물을 출력하는 코드를 작성해주세요.

> 예시
>
> ```js
> case 'tiger':
>    console.log('호랑이');
>    break;
> ```
